# Sol Lens — Tính năng

## Ví
- Tạo ví mới hoặc import từ mnemonic
- Mở khoá bằng mật khẩu mỗi lần khởi động
- Xem số dư SOL + token đang giữ
- Export ví để backup

## Feed token
- Danh sách token mới xuất hiện liên tục theo thời gian thực
- Mỗi token có điểm an toàn (0–100), giá, market cap, thanh khoản, tuổi
- Bộ lọc: 3 preset nhanh (Degen / Balanced / Safe) hoặc tự chỉnh các tiêu chí
- Cài đặt bộ lọc được lưu lại

## Chi tiết token
- Số liệu: giá, market cap, volume, số holder
- Phân tích điểm an toàn theo từng tiêu chí
- Biểu đồ giá

## Trade
- Chọn số SOL và slippage, xem quote trước khi mua
- Bấm Buy → token vào portfolio ngay

## Capybara Pet
- Con capybara nổi trên màn hình
- Khi có token mới → hiện bong bóng thông báo
- Hover vào → mua nhanh không cần vào cửa sổ chính

## Portfolio
- Danh sách vị thế đang mở, tự cập nhật PnL
- Đặt stop loss per token hoặc toàn danh mục, tự động bán khi chạm ngưỡng
- Bán thủ công bất kỳ lúc nào
- Lịch sử trade đã đóng
